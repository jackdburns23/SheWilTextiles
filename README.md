# SheWilTextiles Landing Page Lab

This repo is a clean learning space for practicing landing page design in TypeScript/TSX.

## Goal

Learn how to build a clean landing page using a repeatable structure:

Hero section
Services section
Photo grid
Testimonials
Call to action

## Next Step

Run:

npx create-next-app@latest .

Select TypeScript + Tailwind.

## Structure

app/
components/
data/
public/photos/
docs/
references/

## Rule

Build section by section. Do NOT try to build the full site at once.
